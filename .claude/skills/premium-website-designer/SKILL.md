# Premium Website Designer

-# Instructions

--# Step 1: Engineer the First Impression (The Halo Effect)
When starting a design, obsess over the top half of the homepage (the hero section) to establish a positive cognitive bias within the first 50 milliseconds [6, 9]. 
- Define a single emotion you want the visitor to feel immediately (e.g., calm, confidence, or excitement) [9, 10].
- Keep the layout clean, professional, and uncluttered, using high-quality imagery and a single bold headline [11, 12]. 
- Avoid cheap-looking headers to prevent a negative lens of skepticism [11].

--# Step 2: Declare War on Cognitive Load
Ensure the website feels simple, clear, and intuitive to create "cognitive fluency" [7]. Our brains are inherently lazy, so reducing mental effort makes the brand feel more trustworthy and professional [7, 11].
- Generously apply whitespace to give content room to breathe; treat it as a psychological cue of exclusivity and calm rather than empty space [7, 9, 12].
- Create a clear visual hierarchy with exactly one primary goal for each section [7, 10].
- Simplify the navigation making it predictable, and rigorously remove unnecessary elements [7, 10].

--# Step 3: Hunt for Micro-Interaction Opportunities
Incorporate tiny, satisfying details to leverage the "peak-end rule" and spark joy for the user [8]. Premium sites feel alive because of these thoughtful details that signal craftsmanship and care [12].
- Specify subtle hover animations, like buttons gently changing color [8].
- Add feedback loops, such as gentle confirmations when form fields are filled out correctly [8].
- Include smooth transitions as the user scrolls down the page or opens menus [8, 12].

Examples

Example 1: Luxury Fashion Homepage
User says: "Design a premium landing page for a new high-end bag company."
Actions:
1. Engineer First Impression: Propose a minimalist hero section with extreme whitespace, a stunning fullscreen product shot, and zero clutter to signal confidence [9, 12].
2. Reduce Cognitive Load: Strip down the navigation menu to just three essential links and focus attention purely on the product [9, 10].
3. Add Micro-interactions: Specify a smooth page-load transition and a satisfying visual confirmation when adding an item to the cart [8, 10].
Result: A detailed layout and interaction spec that feels exclusive, calm, and highly professional.

Troubleshooting

Error: The website feels chaotic, stressful, or cheap.
Cause: The design creates a high cognitive load due to competing elements or a cluttered first impression [7, 11].
Solution: Increase whitespace, simplify the navigation, remove secondary calls to action, and ensure the visual hierarchy clearly guides the eye to a single focus point [7, 10].
